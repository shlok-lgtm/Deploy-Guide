# Basis Protocol вЂ” Replit Deployment Guide

## Prerequisites

You need:
- A Replit account (free or paid)
- A Neon PostgreSQL database (free tier works: https://neon.tech)
- A CoinGecko Pro API key ($0/month demo tier: https://www.coingecko.com/en/api)
- An Etherscan API key (free: https://etherscan.io/apis)

---

## Step 1: Create the Repl

1. Go to https://replit.com and click **Create Repl**
2. Choose **Python** as the template
3. Name it `basis-protocol` (or whatever you want)
4. Click **Create Repl**

## Step 3: Set Environment Variables (Secrets)

In Replit, click the **Secrets** tab (lock icon in the left sidebar).

Add these secrets:

| Key | Value | Required |
|-----|-------|----------|
| `DATABASE_URL` | Your Neon PostgreSQL connection string | **Yes** |
| `COINGECKO_API_KEY` | Your CoinGecko API key | **Yes** |
| `ETHERSCAN_API_KEY` | Your Etherscan API key | **Yes** |
| `ANTHROPIC_API_KEY` | Your Anthropic API key (for content drafts) | Optional |
| `WORKER_ENABLED` | `true` | Optional (defaults to true) |
| `COLLECTION_INTERVAL` | `60` | Optional (minutes between scoring cycles) |

Your Neon DATABASE_URL looks like:
```
postgresql://username:password@ep-xyz-12345.us-east-2.aws.neon.tech/neondb?sslmode=require
```

## Step 4: Configure the Run Command

Click the three dots next to the Run button (or open `.replit` file) and set:

**Run command:**
```
python main.py
```

If Replit asks you to configure the run command, just enter `python main.py`.

If there's a `.replit` file, make sure it contains:
```toml
run = "python main.py"
entrypoint = "main.py"

[nix]
channel = "stable-24_05"
```

## Step 5: Install Dependencies

Open the Shell tab and run:

## Step 9: Verify

Open these URLs in your browser (replace with your Replit URL):

| Endpoint | What it shows |
|----------|--------------|
| `https://your-repl.replit.app/api/health` | Database connection status |
| `https://your-repl.replit.app/api/stablecoins` | List of tracked stablecoins |
| `https://your-repl.replit.app/api/scores/latest` | Latest SII scores (empty until first worker cycle) |
| `https://your-repl.replit.app/api/governance/stats` | Governance intel stats |
| `https://your-repl.replit.app/api/governance/debates?days=30` | Hot governance debates |
| `https://your-repl.replit.app/api/content/digest?days=14` | Content opportunities |

The first scoring cycle takes ~2-3 minutes (hits CoinGecko, DeFiLlama, Curve, Etherscan). After that, scores appear at `/api/scores/latest`.

## Step 10: (Optional) Manual Governance Crawl

To trigger a fresh governance crawl immediately:
```bash
# In Shell:
python -c "from app.governance import run_crawl; run_crawl(since_days=7)"
```

Or via the API:
```bash
curl -X POST https://your-repl.replit.app/api/governance/crawl
```

The worker also auto-crawls governance forums every ~6 hours.

## API Reference (Quick)

### SII Scores
- `GET /api/scores/latest` вЂ” Current scores for all stablecoins
- `GET /api/scores/{coin_id}` вЂ” Single coin detail
- `GET /api/scores/history/{coin_id}?days=30` вЂ” Score history

### Governance Intel
- `GET /api/governance/stats` вЂ” Document counts, top stablecoins
- `GET /api/governance/debates?days=7` вЂ” Hot debates
- `GET /api/governance/sentiment?days=14` вЂ” Sentiment by stablecoin
- `GET /api/governance/metrics?days=7` вЂ” What risk metrics people are discussing

### Content Engine
- `GET /api/content/digest?days=7` вЂ” Ranked content opportunities
- `POST /api/content/draft` вЂ” Generate a draft thread from a signal
- `GET /api/content/arc/{day}` вЂ” 60-day content arc entry

## Troubleshooting

**"Database connection failed"**
в†’ Check your DATABASE_URL in Secrets. Make sure it includes `?sslmode=require` for Neon.

**"No module named 'psycopg2'"**
в†’ Run `pip install -r requirements.txt` in Shell.

**Scores are empty**
в†’ Wait 2-3 minutes for the first worker cycle. Check the console logs for API errors (usually means missing CoinGecko key).

**Governance data empty**
в†’ Run the import: `python import_governance.py --sql-file exports/governance_export.sql`

**Port already in use**
в†’ Stop the current run, wait 10 seconds, try again. Or set `API_PORT=3000` in Secrets.
